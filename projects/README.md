# In-Repo Projects

Flagship tools maintained directly in this repository. Each project has its own folder with a standalone README.

| Project | Description |
|---------|-------------|
| [te-questionnaire-app](te-questionnaire-app/) | RFP/security questionnaire automator using ThousandEyes docs and local Ollama |

## Adding a project here

1. Create `projects/your-tool/` with source code and README
2. Add an entry to [`docs/data/projects.json`](../docs/data/projects.json)
3. Follow the [contributing guide](../CONTRIBUTING.md) — no secrets, no internal URLs

External-only projects can live in separate repos and be linked from the catalog without a folder here.
