# CLAUDE.md — GES & ASP SE Hub

This repo includes agent workflows for contributing to the team hub.

## Share with the group

When the user says **"Let's share with the group"**, follow the full workflow in:

`.cursor/skills/share-with-group/SKILL.md`

Read `.cursor/skills/share-with-group/reference.md` for JSON templates, public-safe checklist, Discussion/Webex copy, and preview conventions.

**Do not** edit `docs/data/projects.json` or push until mockup approval and explicit publish confirmation.

## HTML presentation deck

When the user says **"I want to make a deck"** (or similar), follow the full workflow in:

`.cursor/skills/html-deck/SKILL.md`

Read `.cursor/skills/html-deck/reference.md` for slide components, brand rules, and ThousandEyes MCP integration.

**Do not** generate deck HTML until intake is complete and (when a customer is named) the user confirms customer context.

## Hub layout

| Path | Purpose |
|------|---------|
| `docs/` | GitHub Pages site (live hub) |
| `docs/data/projects.json` | Project catalog |
| `docs/share-previews/` | Pre-publish HTML mockups |
| `projects/` | In-repo flagship tools |

## Public-safe rule

Never include internal hostnames, credentials, or customer-identifiable data in catalog entries or previews.
